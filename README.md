# Проект: Mesto Russia

**About a project** - проект для загрузки фотографий красивых мест России 

## Функционал

1. Редактирования имени и должности автора страницы;
2. Добавление изображений в избранное;

## Технологии

  - БЭМ
  - flexbox
  - Grid layout
  - normalize.css
  - JS
